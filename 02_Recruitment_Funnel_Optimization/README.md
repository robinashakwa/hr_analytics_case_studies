# Recruitment Funnel Optimization with Source ROI

## Business Problem
HR leadership wants to understand which sourcing channels and roles
produce the fastest hires and the best long-term performers, in order
to optimize recruitment spend.

## Objective
- Analyze recruitment funnel conversion by source and role
- Measure time-to-hire efficiency
- Evaluate cost-per-hire and source ROI

## Tools
- SQL (funnel and ROI logic)
- Excel (cohort and ROI validation)
- Power BI (funnel and ROI dashboards)

## Status
SQL logic in progress.
